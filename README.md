### Análisis exploratorio del dataset Adult Income

Exploración de variables sociodemográficas asociadas con ingresos superiores a 50.000 dólares.

### Análisis exploratorio del dataset Wine Quality

Exploración de las características fisicoquímicas asociadas con la calidad del vino en el dataset Wine Quality de UCI.

[Ver repositorio del proyecto](https://github.com/francorte/eda-wine-quality-uci)
